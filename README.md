## Technologies:

* Ruby 2.6.3
* Vue
* Tailwind CSS


### Setup

Navigate to directory where the code script is located

```sh
$ cd ~/cran_indexer
```
Install the dependencies and devDependencies.

```sh
$ rvm use 2.6.3
````
```
$ bundle install
```

```
$ yarn install
```

```
$ rails db:setup
```


### Steps to run Application

```sh
$ foreman start
```
